# Python API reference

The primary domain objects, generated from their docstrings. `SnowDb` is the
read/query surface; `SnowDbManager` wraps it with the admin/write surface. A
`Dataset` is one gridded dataset within a snowdb, described by a `DatasetSpec`.

This is a curated starting selection of the public surface — extend it as more
of the library becomes a supported entry point.
